# AirlockFriends
Friending system for Among Us 3D / Among Us 3D: VR, you can friend others with the mod to then message, join, invite, request to join, and more.

# Join the Discord please, for support, early release, testing, make friends, or if you just want to talk to others who play AU3D.
# https://discord.gg/UHGQyPT7ts

# Info
This is a legal AU3D mod that works for the 3D and VR version of Among Us VR, when you send a friend request to someone using their friend code and they accept, you both are able to message, join request, invite each other and more.
All these are able to be disabled in settings, you can disable friend requests, invites, join requests, messages, everything really.
You can toggle the UI by clicking F1 or /

# Will I get banned?
No, this is a legal mod that follows AU3D modding policy at https://www.innersloth.com/among-us-mod-policy/

# Is this malware?
No, it is open sourced and trusted by many others, feel free to look through the files yourself!
And all requests (friend requests, invites, everything) are handled on my server, so don't worry about people exploiting it to hurt your game.

# How do I use it?
If you do not have melonloader installed, simply download the zip in the latest release and extract it, then move all files from there to your Among Us 3D or VR folder
If you DO have melonloader, simply just download the AirlockFriends.dll and move it into your mods folder
